## 1. 美股数据源调研：yfinance

### 可行性结论：完全可行

yfinance 当前版本 v1.2.0（2026-02），已从 v0.2 稳步升级到 v1.0 正式版，维护活跃。

**ETF 支持情况：**

| ETF | 名称 | AUM | 日均成交量 | 状态 |
|-----|------|-----|-----------|------|
| XLK | Technology Select Sector SPDR | 大型 | 高 | 正常 |
| SOXX | iShares Semiconductor ETF | 大型 | 高 | 正常 |
| **CARZ** | First Trust Future Vehicles ETF | **~$49M** | **~5,545** | **流动性极差** |
| XLV | Health Care Select Sector SPDR | 大型 | 高 | 正常 |
| XLF | Financial Select Sector SPDR | 大型 | 高 | 正常 |
| XLE | Energy Select Sector SPDR | 大型 | 高 | 正常 |
| XLY | Consumer Discretionary SPDR | 大型 | 高 | 正常 |
| XLRE | Real Estate SPDR | 大型 | 高 | 正常 |
| XLB | Materials Select Sector SPDR | 大型 | 高 | 正常 |
| XLC | Communication Services SPDR | 大型 | 高 | 正常 |
| XLI | Industrial Select Sector SPDR | 大型 | 高 | 正常 |
| XLU | Utilities Select Sector SPDR | 大型 | 高 | 正常 |
| THNQ | Robo Global AI ETF | ~$288M | ~16,525 | 正常 |
| ICLN | iShares Global Clean Energy | ~$2.1B | ~4.6M | 正常 |
| GLD | SPDR Gold Shares | 大型 | 高 | 正常 |

### 关键发现

1. **CARZ 流动性严重不足**：日均成交量仅 ~5,545 股，买卖价差高达 ~20%，价格数据质量不可靠
   - **建议替换为 DRIV**（Global X Autonomous & Electric Vehicles ETF，AUM ~$350M，日成交量 10万+）
2. **速率限制**：对于 15 个 ETF 的每日批量获取，使用 `yf.download()` 一次请求完成，完全在限制范围内
3. **复权价格**：默认 `auto_adjust=True`，Close 列即为复权后收盘价，直接用于涨跌幅计算
4. **v1.2.0 注意**：`history()` 返回的 DataFrame 默认只读，修改前需 `.copy()`

### 推荐用法

```python
import yfinance as yf

tickers = "XLK SOXX DRIV XLV XLF XLE XLY XLRE XLB XLC XLI XLU THNQ ICLN GLD"
data = yf.download(tickers=tickers, start="2025-01-01", auto_adjust=True, group_by="ticker")
```

## 2. A股数据源调研：AKShare

### 可行性结论：可行，需注意接口稳定性

AKShare 是免费开源 Python 金融数据接口库，无需注册、无需 API Key。

### 核心接口

| 功能 | 函数名 | 数据来源 |
|------|--------|---------|
| 申万行业指数历史行情 | `sw_index_daily(symbol="801010", start_date="20240101", end_date="20241231")` | 申万宏源官网 |
| ETF 历史行情 | `fund_etf_hist_em(symbol="512480", period="daily", start_date="20240101", end_date="20241231")` | 东方财富 |
| 板块行业指数列表 | `stock_board_industry_index_name_em()` | 东方财富 |

### 注意事项

1. AKShare 函数名可能随版本更新变化，需锁定版本号（`pip install akshare==x.x.x`）
2. 申万指数接口基于网页抓取，偶有接口失效需等待修复
3. **备选方案**：若 AKShare 申万接口不稳定，可使用 `index_zh_a_hist`（东方财富通用指数接口）或 baostock 库
4. 数据更新时间：A股收盘后（15:00）数据即更新完毕

### 备选数据源对比

| 数据源 | 注册要求 | 稳定性 | 免费 | 适合场景 |
|--------|---------|--------|------|---------|
| AKShare | 无 | 中（依赖爬虫） | 是 | MVP 首选 |
| tushare | 需注册积分 | 高 | 基础功能免费 | 稳定性要求高 |
| baostock | 无 | 高 | 是 | 备选方案 |
| efinance | 无 | 中 | 是 | AKShare 替代 |

## 3. GitHub Actions 定时任务调研

### 可行性结论：完全可行

**2026年3月重大更新**：GitHub Actions 已原生支持 IANA 时区，不再需要手动换算 UTC。

### 配置方案

```yaml
on:
  schedule:
    - cron: '0 5 * * 1-5'  # 北京时间周一至周五 05:00
      timezone: 'Asia/Shanghai'
```

如果使用旧版语法（UTC 方式）：
```yaml
on:
  schedule:
    - cron: '0 21 * * 0-4'  # UTC 周日至周四 21:00 = 北京时间周一至周五 05:00
```

### 资源限制

| 限制项 | 值 |
|--------|---|
| 免费额度（私有仓库） | 2,000 分钟/月 |
| 免费额度（公开仓库） | **无限制** |
| 单次任务最大执行时间 | 6 小时 |
| 预估单次执行时间 | ~2-3 分钟 |
| 月预估消耗 | ~60 分钟（20 交易日） |

### 工作流设计

```
1. Python 环境设置（~30s）
2. pip install 依赖（~30s）
3. 拉取美股 15 个 ETF 数据（~5s）
4. 拉取 A股 15 个行业指数 + ETF 数据（~15s）
5. 计算相关性 + 条件概率（~5s）
6. 输出 JSON 到 data/results/
7. git commit + push（自动触发 GitHub Pages 部署）
```

## 4. 前端热力图方案调研

### 方案对比

| 方案 | 包大小(gzip) | 热力图支持 | 移动端适配 | 易用性 | 推荐度 |
|------|-------------|-----------|-----------|--------|--------|
| **ECharts（按需引入）** | ~100-200KB | 原生支持 | 优秀 | 高 | **推荐** |
| Chart.js + chartjs-chart-matrix | ~140KB | 需插件 | 良好 | 中 | 备选 |
| ApexCharts | ~140KB | 原生支持 | 良好 | 高 | 备选 |
| Plotly.js | ~1MB+ | 原生支持 | 偏重 | 高 | 不推荐 |
| 纯 CSS/Canvas 实现 | ~5KB | 自定义 | 优秀 | 低 | 最轻量 |

### 推荐方案：ECharts 按需引入

**理由：**
1. 热力图是原生图表类型，无需额外插件
2. 15x15 矩阵在移动端渲染性能优秀
3. 支持点击交互（点击热力图单元格查看详情）
4. 中文文档完善，社区活跃
5. 按需引入可将包体积控制在 ~150KB 以内

**替代方案（更轻量）：** 如果对包体积极度敏感，可以考虑纯 CSS Grid + 背景色实现热力图（~5KB），牺牲部分交互体验换取极致性能。

## 5. 数据合规与法律调研

### 核心结论：无需牌照，但需注意措辞

**不需要证券投资咨询牌照**，因为：
- 展示板块间统计相关性，非个股推荐
- 计算统计指标（相关系数、条件概率），非价格预测
- 免费工具，无付费门槛
- 不提供买入/卖出信号

### 需要调整的措辞

| 原始设计 | 风险 | 建议修改 |
|---------|------|---------|
| "异动信号" | 暗示可操作的交易信号 | **"显著波动"** |
| "美股异动 x A股机会" | "机会"暗示投资建议 | **"隔夜映射 · 跨市场相关性分析"** |
| 信号卡片排序逻辑 | 暗示投资优先级 | 保持，但标注"按统计显著性排序" |

### 必需免责声明要素

1. **非投资建议**：本工具仅供学术研究和数据参考，不构成任何投资建议
2. **数据准确性**：数据来源于第三方公开接口，准确性不做保证
3. **风险提示**：股市有风险，投资需谨慎
4. **非持牌声明**：本工具运营方非持牌证券投资咨询机构
5. **统计性质**：相关性系数为历史统计结果，不代表未来走势
6. **数据来源**：明确标注 Yahoo Finance 和公开市场数据

### 数据源合规

| 数据源 | 许可证 | 风险等级 | 说明 |
|--------|--------|---------|------|
| yfinance | Apache 2.0 | 低 | Yahoo 数据官方声明"仅供个人使用"，但小型研究工具通常被容忍 |
| AKShare | MIT | 低 | 抓取公开网站数据，免费非商业用途通常被容忍 |

### 隐私合规

- 工具不收集用户数据（无账户系统）
- 无需担心《个人信息保护法》（PIPL）合规
- GitHub Pages 托管，无需 ICP 备案
