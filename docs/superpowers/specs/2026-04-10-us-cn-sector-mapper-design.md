# US-CN Sector Mapper - MVP Design

> 2026-04-10 | 独立新项目

## 定位

数据分析工具：追踪美股板块隔夜异动，通过统计相关性分析，展示次日 A 股对标板块及 ETF 的可能反应。面向散户投资者，提供数据参考，不构成投资建议。

## 核心逻辑

美股在北京时间 21:30 - 次日 04:00 交易，A 股 09:30 开盘，存在约 5.5 小时的信息传递窗口。通过历史数据计算美股板块涨跌与 A 股板块次日开盘的统计相关性，为用户提供跨市场参考。

每个 A 股板块关联 **1 个申万行业指数 + 1-2 只流动性最好的 ETF**。

### 统计指标

对每对板块计算：

- **隔夜相关系数**：美股当日涨跌幅 vs A 股次日开盘涨跌幅的 Pearson 相关系数
- **滚动窗口**：30 日 / 60 日 / 120 日三个周期
- **条件概率**：当美股板块涨跌幅 > 2% 时，A 股对标板块次日高开（>0%）的概率
- **平均影响幅度**：条件触发后 A 股板块的平均涨跌幅

## 系统架构

```
┌─────────────────────────────────────────────┐
│              定时任务层                        │
│  cron: 每天美股收盘后(北京时间 05:00)执行       │
├──────────┬──────────┬───────────────────────┤
│ 数据采集  │ 相关性计算 │ 结果存储              │
│ yfinance │ pandas   │ SQLite / JSON         │
│ AKShare  │ numpy    │                       │
├──────────┴──────────┴───────────────────────┤
│              数据层 (Python)                   │
│  data/us_sectors/    美股板块日线              │
│  data/cn_sectors/    A股板块+ETF日线           │
│  data/correlations/  相关性计算结果            │
├─────────────────────────────────────────────┤
│              展示层 (H5)                       │
│  Python 脚本输出 JSON → 前端 fetch 展示        │
│  手机端适配，卡片式布局                         │
└─────────────────────────────────────────────┘
```

### 数据流

```
美股收盘(~05:00 北京时间)
  → Python 脚本拉取当日美股板块涨跌幅
  → 拉取 A 股昨日收盘数据（用于滚动计算）
  → 更新相关性矩阵（30/60/120日窗口）
  → 输出 JSON 结果文件
  → H5 前端读取展示
```

## 技术选型

| 层 | 选择 | 理由 |
|---|------|------|
| 数据采集 | yfinance + AKShare | 免费，Python 生态成熟 |
| 数据分析 | pandas + numpy + scipy | 标准数据科学生态 |
| 数据存储 | JSON 文件 + SQLite | MVP 无需数据库服务 |
| 后端 | Python 脚本 + GitHub Actions 定时执行 | 零服务器成本 |
| 前端 | 原生 HTML/CSS/JS（Vite） | 移动端适配，经验复用 |

关键决策：后端用 GitHub Actions 定时跑 Python 脚本，输出 JSON 到仓库，前端直接读取——零服务器成本。

## H5 页面设计

### 页面结构（单页为主）

1. **Header**：标题「隔夜映射」+ 副标题「美股异动 x A股机会」
2. **日期选择器**：选择查看日期
3. **信号卡片列表**：按「条件概率 x 美股涨跌幅绝对值」降序排列
   - 美股板块名 + 涨跌幅
   - 箭头指向 A 股对标板块
   - 条件概率 + 平均影响幅度
   - 关联 A 股 ETF 代码
   - 可展开：30/60/120 日相关系数 + 样本数
4. **相关性热力图**：颜色深浅表示相关度，横轴美股纵轴 A 股，可点击查看详情
5. **完整板块对照表**：表格，可排序
6. **免责声明**

### 规则

- 异动阈值：美股板块单日涨跌幅 > 2% 标记为「异动信号」
- 颜色规则：涨红跌绿（A 股习惯）
- 最大宽度：480px 居中
- 前端静态读取 JSON，无实时刷新

## MVP 范围

### 做

- 15 对美股 <-> A 股板块映射（含 A 股 ETF）
- 每日自动采集美股板块数据 + A 股板块数据（GitHub Actions）
- 计算 30/60/120 日滚动相关性 + 条件概率
- H5 展示：信号卡片 + 相关性热力图 + 对照表
- 免责声明

### 不做（后续迭代）

- 港股对标（第二期）
- 推送通知（微信/邮件/钉钉）
- 个股级映射
- 用户系统 / 收藏 / 自选
- 回测工具 / 策略模拟
- 付费数据源升级

## 验收标准

- [ ] 能拉取 15 对板块最近 120 个交易日的数据
- [ ] 相关性计算结果正确（抽样验证 3 对）
- [ ] H5 手机端（375px）正常展示信号卡片和热力图
- [ ] GitHub Actions 每日自动执行
- [ ] 美股涨跌 > 2% 的板块正确标记为异动信号
- [ ] 每个信号卡片展示关联 A 股 ETF
- [ ] 含免责声明
